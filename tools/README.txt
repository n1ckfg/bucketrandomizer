Process:

1. Basic workflow
1.1. Run extractor.py on an original odt file to create a new json file.
1.2. Upload the json file to the root of the project website.

2. Fixing formatting errors
2.1. In LibreOffice, open the original odt file.
2.2. In LibreOffice, select all text and copy.
2.3. In Word, create a new blank document.
2.4. In Word, paste the text and save as a docx file.
2.5. In LibreOffice, open the docx file and save as a new odt file. 
2.6. Repeat Part 1.
